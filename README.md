## 8. Manual de instalación y deployment:
```
Clone the repository:
    git clone https://github.com/nadaronda/Mariana-sister

Initiate npm:
    npm init
Or use this instead:
    npm init -y

Install dependencies:
    npm install

Execute the project:
    npm run dev

```
